## 🤖 4. Reddit (r/IndianStreetBets or r/IndiaInvestments)

**Title:** I got tired of paying for clunky FII/DII data, so I built an open-source terminal that pulls directly from NSE. Thought you guys might like it.

Hey guys, 

Tracking FII buying/selling is a pain. The free sites are full of ads, and the premium ones cost a fortune. I wanted something clean, dark-mode, and instant. So I built a single-file terminal. 

It does not have a backend, it does not require you to login or give me your email, and it definitely does not have cookies. It’s a single HTML file that uses a proxy to fetch today's flow data directly from the NSE site when you open it.

**What it does:**
- Shows FII & DII net flows with a strength bar to see who is dominating.
- Has a Streak Tracker to see if FIIs have been consecutively dumping for 5+ days (and how many thousands of crores they've pulled).
- Has 45-day visual heatmaps so you can see the trend visually instead of looking at tables.

**How to run it:**
Literally just download `fii_dii_india_flows_dashboard.html` from the repo and double-click it. It opens in your browser and works automatically. 

**Link:** https://github.com/MrChartist/fii-dii-data

Happy to take feature requests or pull requests if anyone wants to add Nifty overlay charts to it!
